# Create a multi-tenancy application in Nest.js
## Description

Build multi tenants app using [Nest](https://github.com/nestjs/nest) framework and microservices.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start ${serviceName}

# watch mode
$ npm run start:dev ${serviceName}

```
